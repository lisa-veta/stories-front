# stories-front

Фронтенд административной панели управления рекламным контентом для приложения Интерсвязь

## Написание кода

### Структура кода

Код в проекте организован в соответствии с методологией [Feature-Sliced Design](https://feature-sliced.design/ru/docs)

### Проверка линтером

В проекте настроены eslint и stylelint

- `npm run lint:code` - проверка кода

### Локальный запуск

```bash
npm run dev
```
